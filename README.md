# Moai Idle Game (모아이 키우기)

- 유니티로 개발한 2D 방치형 Merge 모바일 게임입니다.
- 동일한 캐릭터를 합성(Merge)하여 성장시키고, 자동으로 골드를 획득하는 구조의 게임입니다.

---

## 📌 프로젝트 개요
- 장르: 2D 방치형 Merge 게임
- 개발 인원: 1인 개발
- 플랫폼: Android
- 개발 도구: Unity 2022.3 LTS

---

## 🛠 사용 기술
- Unity
- C#
- JSON
- Android APK 빌드

---

## 🎮 플레이 방법

### ▶ APK로 실행 (추천)
- 아래 링크에서 APK 다운로드 후 설치
- 안드로이드 기기에서 "알 수 없는 앱 설치" 허용 필요

📥 **APK 다운로드**  
https://github.com/mntjksn/Moai_Idle_Game/releases

---

### ▶ Unity 프로젝트로 실행
1. GitHub 레포지토리 클론 또는 다운로드
2. Unity Hub에서 프로젝트 추가
3. Unity 2022.3 LTS로 열기
4. 패키지 로딩 완료 후 Play 실행

---

## ✨ 주요 특징
- 방치형 게임 구조를 기반으로 한 자동 골드 획득 시스템
- 캐릭터 수집 및 성장 요소 구현
- 모바일 환경에서 플레이 가능하도록 구성
